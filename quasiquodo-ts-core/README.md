# quasiquodo-ts-core

This crate is an internal implementation detail of [**quasiquodo-ts**](https://crates.io/crates/quasiquodo-ts). It contains the core logic for preprocessing and parsing TypeScript source literals, and generating Rust code to construct AST nodes.

**Please use quasiquodo-ts directly.** This crate's API is not stable, and may change without notice.
