# quasiquodo-py-core

This crate is an internal implementation detail of [**quasiquodo-py**](https://github.com/linabutler/quasiquodo/tree/main/quasiquodo-py). It contains the core logic for preprocessing and parsing Python source literals, and generating Rust code to construct AST nodes.

**Please use quasiquodo-py directly.** This crate's API is not stable, and may change without notice.
